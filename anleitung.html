Anleitung

Willkommen zur Quiz-Anleitung!

Dieses Quiz ist so aufgebaut, dass Sie über einen Excel-Report Fragen importieren und anschließend in einem spannenden Spielverlauf als Teams gegeneinander antreten können. Im Folgenden finden Sie alle wichtigen Schritte und Regeln:

  1. Excel-Template herunterladen und bearbeiten:
    • Laden Sie zunächst das Excel-Template herunter, indem Sie auf den Button "Excel-Template herunterladen" in der Aktionsleiste klicken.
    • Öffnen Sie die Vorlage in Excel. Das Template ist in verschiedene Spalten unterteilt, die jeweils einer Kategorie zugeordnet sind: easy, medium, hard und death.
    • In jeder Spalte tragen Sie Ihre Fragen ein. Jede Frage besteht aus drei Teilen:
      • Ein Header (z. B. "Easy Frage 1"), der den Schwierigkeitsgrad und die Frage-Nummer angibt.
      • Die eigentliche Frage.
      • Eine ausführliche Erklärung oder Antwort.
    • Zwischen den einzelnen Fragen sollte ein Leerblock vorhanden sein (d.h. nach jeweils vier Zeilen: Header, Frage, Erklärung und Leerzeile), damit der Import die Daten korrekt zuordnen kann.
  2. Excel-Report importieren:
    • Nachdem Sie die Vorlage bearbeitet und gespeichert haben, klicken Sie auf den Button "Excel Import 🗂️" in der oberen Aktionsleiste.
    • Wählen Sie die bearbeitete Excel-Datei aus. Das System importiert daraufhin alle Fragen aus der Datei in den globalen Fragenpool.
    • Nach einem erfolgreichen Import wird automatisch die Team-Auswahl eingeblendet.
  3. Team-Auswahl und Spielstart:
    • Wählen Sie über das Dropdown-Menü die Anzahl der teilnehmenden Teams aus.
    • Starten Sie das Spiel, indem Sie auf den Button "Spiel starten" klicken.
    • Die Teams werden nun im Spielbereich angezeigt und das Fragen-Gitter wird sichtbar.
  4. Fragen beantworten:
    • Klicken Sie auf eine Frage im Fragen-Gitter, um diese anzuzeigen.
    • Ein Timer startet automatisch, der je nach Schwierigkeitsgrad variiert (z. B. 60 Sekunden für easy, 120 Sekunden für medium usw.).
    • Innerhalb des Zeitlimits müssen Sie die Frage beantworten. Sie können über den Button "Antwort anzeigen" die Erklärung einblenden.
    • Anschließend bestätigen Sie das Ergebnis mit den Buttons "Richtig ✅" oder "Falsch ❌".
  5. Punktevergabe und Wiederwahl:
    • Bei einer richtigen Antwort erhält das Team die volle Punktzahl der Frage.
    • Bei einer falschen Antwort wird der Versuchszähler erhöht und zukünftige Punkte für diese Frage werden halbiert.
    • Nach dem ersten Fehlversuch bleibt die Frage weiterhin auswählbar, allerdings nur noch mit halber Punktewertung. Erst nach zwei Fehlversuchen wird die Frage als verwendet markiert und kann nicht mehr gewählt werden.
  6. Spielende:
    • Das Spiel endet, wenn alle Fragen aus allen Kategorien verwendet wurden.
    • Das Team mit den meisten Punkten wird als Sieger bekannt gegeben.

Viel Erfolg und vor allem viel Spaß beim Quiz!
